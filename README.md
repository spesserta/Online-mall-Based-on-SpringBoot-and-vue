# 大三的期末大作业——仿照某网上购物平台创建的基于SpringBoot和Vue的在线商城❤️<br>


代码文件在master分支<br>


>管理员账号：admin  密码：admin

* 开发语言：Java
* 开发框架：SpringBoot + vue + jsp + css
* JDK版本：JDK 1.8
* 服务器：本地tomcat7
* 数据库：mysql
* 数据库工具：Navicat
* 开发软件：IDEA + VScode
* Maven包：Maven3.3.9
* 浏览器：Edge
